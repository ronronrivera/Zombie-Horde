# Credits

ZombieHorde is built on the following open source libraries and assets.

---

## Libraries

### GLAD
- **Author**: David Herberth
- **License**: MIT
- **Source**: https://glad.dav1d.de
- OpenGL loader generator. Used to load all OpenGL function pointers at runtime.

### GLFW
- **Author**: Marcus Geelnard, Camilla Löwy
- **License**: zlib/libpng
- **Source**: https://www.glfw.org
- Window creation, OpenGL context management, keyboard and mouse input.

### GLM (OpenGL Mathematics)
- **Author**: G-Truc Creation
- **License**: MIT
- **Source**: https://github.com/g-truc/glm
- Header-only C++ mathematics library for graphics. Used for vectors, matrices, and transforms.

### stb_image
- **Author**: Sean Barrett
- **License**: MIT / Public Domain
- **Source**: https://github.com/nothings/stb
- Single-header image loader. Used to load PNG textures from disk.
